# Fire Safety Systems - Incident Runbooks

## 🚨 CRITICAL: Fire Alarm Activated

### Severity: P1 - Critical
### Response Time: Immediate (< 1 minute)

#### Symptoms
- Fire alarm sounding
- Smoke/heat detector triggered
- Manual pull station activated
- Fire suppression system armed/discharged

#### Immediate Actions (First 60 Seconds)
1. **CALL 000 IMMEDIATELY** - Fire Brigade
2. **Acknowledge incident** in PagerDuty
3. **Initiate evacuation** per emergency procedures
4. **Account for all personnel**
5. **DO NOT re-enter building**

#### Investigation Steps (ONLY if safe and authorized)
1. **Check fire panel**
   - Which zone triggered?
   - Single or multiple detectors?
   - Type of detector (smoke/heat)?
2. **Visual inspection** (if safe)
   - Visible fire or smoke?
   - Location of activation
   - Severity assessment

#### Resolution Steps

### ⚠️ IF ACTUAL FIRE CONFIRMED:
1. **Evacuate immediately**
2. **Call 000** if not already done
3. **Close doors** to contain fire
4. **Do not use elevators**
5. **Meet at assembly point**
6. **Wait for Fire Brigade**
7. **Do not re-enter until cleared by Fire Brigade**

### ⚠️ IF FALSE ALARM (Fire Brigade confirms):
1. **Silence alarm** (authorized personnel only)
2. **Reset fire panel**
3. **Document cause of false alarm**
4. **Inspect detector that triggered**
5. **Test fire system**
6. **Return to normal operations**

### ⚠️ IF SUPPRESSION SYSTEM DISCHARGED:
1. **Ensure area is safe** (Fire Brigade clearance)
2. **Ventilate area** before entry
3. **Inspect all equipment for damage**
4. **Document discharge event**
5. **Contact suppression system service provider**
6. **Schedule system recharge**
7. **Test equipment before returning to service**
8. **File insurance claim**

#### Post-Incident (Mandatory)
- [ ] **File incident report with Fire Brigade**
- [ ] **Document complete timeline**
- [ ] **Preserve all evidence**
- [ ] **Conduct mandatory post-mortem**
- [ ] **Review evacuation procedures**
- [ ] **Test all fire safety systems**
- [ ] **Notify insurance company**
- [ ] **Customer notification if service impacted**
- [ ] **Update emergency procedures**
- [ ] **Staff debriefing and training**

#### Escalation
- **Immediate:** Call 000 (Fire Brigade)
- **Immediate:** Notify all staff to evacuate
- **Immediate:** Notify Facilities Manager and Director
- **Within 1 hour:** Notify insurance company
- **Within 4 hours:** Customer notification if impacted

#### Critical Contacts
- **Emergency Services:** 000
- **Fire Brigade Non-Emergency:** [Local number]
- **Fire System Service Provider:** [24/7 number]
- **Insurance Company:** [Emergency claims number]
- **Building Management:** [Emergency contact]

## ⚠️ HIGH: Fire Suppression System Fault

### Severity: P2 - High
### Response Time: < 5 minutes

#### Symptoms
- Suppression system offline
- Low pressure alarm
- Control panel fault
- No fire protection available

#### Immediate Actions
1. **Acknowledge incident**
2. **Verify system status**
   - Check fire panel
   - Inspect suppression system
   - Confirm fault condition
3. **Implement fire watch**
   - Post fire watch personnel
   - Increase monitoring
   - Prepare portable extinguishers
4. **Contact service provider**
   - Report fault
   - Request emergency service
   - Get ETA for repair

#### Investigation Steps
1. **Identify fault type**
   - Pressure loss?
   - Control system failure?
   - Power supply issue?
   - Sensor malfunction?
2. **Check system history**
   - Recent maintenance?
   - Previous faults?
   - Scheduled testing?

#### Resolution Steps
1. **Immediate:**
   - Implement continuous fire watch
   - Ensure fire extinguishers accessible
   - Brief staff on heightened fire risk
   - Consider hot work permit suspension
2. **Service provider:**
   - Emergency service call
   - Temporary system if available
   - Expedite permanent repair
3. **Until repaired:**
   - Maintain fire watch 24/7
   - Increase inspection frequency
   - Document all fire watch activities
   - Notify Fire Brigade of impaired system

#### Post-Incident
- [ ] Document fault and repair
- [ ] Test system thoroughly
- [ ] Update maintenance schedule
- [ ] Review fire safety procedures
- [ ] File report with Fire Brigade if required

## ⚡ MEDIUM: Fire Extinguisher Inspection Overdue

### Severity: P3 - Medium
### Response Time: < 1 day

#### Symptoms
- Annual inspection date passed
- Compliance requirement not met
- Potential regulatory violation

#### Actions
1. **Schedule inspection immediately**
2. **Contact certified inspector**
3. **Verify all extinguisher locations**
4. **Check for any damaged units**
5. **Update inspection tags**
6. **Document compliance**

---

## ℹ️ LOW: Fire Drill Scheduled

### Severity: P4 - Low
### Response Time: < 1 week

#### Actions
1. Schedule fire drill with all stakeholders
2. Notify staff and customers
3. Coordinate with Fire Brigade if required
4. Prepare drill scenario
5. Conduct drill and document results
6. Review and update procedures
